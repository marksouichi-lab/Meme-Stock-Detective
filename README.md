Write a short GitHub README for my AI app. ~100 words.
Meme Stock Detective is a dark-themed web app that tracks sentiment around popular "meme stocks" (GME, AMC, TSLA, NVDA, etc.). For each ticker, it pulls recent Yahoo Finance news headlines and scores them with a natural-language sentiment analyzer to gauge whether coverage is bullish, neutral, or bearish. It layers that with real price data — current price, and average price on up-days vs. down-days — to approximate how "longs" and "shorts" in the community are positioned. The homepage lets you search any ticker for an instant sentiment/risk readout, browse trending meme stocks, and view a full company table with sentiment badges and a "Market Mood" summary showing the overall bullish/neutral/bearish split across the whole watchlist.
